# RnB4u.in

A simple music scraper for new songs.

## Install Requirements

Install the project's dependencies with:

`pip3 install -r requirements.txt`

## Scrape

Fire up the bot:

`python3 rnb4u.py`

## New Songs

Open [music.txt](music.txt) for mp3 links
